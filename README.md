# WeHo Temps United

This repository contains the active static website for **WeHo Temps United** (wehotempsunited.org). 

Originally built on WordPress and hosted on Pantheon, this site was migrated to a static HTML version in December 2025 to ensure performance and reliability during the active unionization campaign.
